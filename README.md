# 项目描述

* 此项目为新闻资讯的 SPA Web App
* PC端, 移动端 双端自适应
* 功能模块包括: 首页, 新闻详情, 新闻评论, 个人中心, 用户注册/登陆
* 使用 React + ES6 + Webpack + Babel 等前端最新最热的技术
* 用户界面主要使用蚂蚁金服基于 React 的 UI 库: Ant Design
* 项目采用模块化、组件化、工程化的模式开发

# 运行项目

## 克隆项目

`https://github.com/huajianduzhuo/news-react.git`

## 安装依赖

`npm install`

## 运行项目

`npm start`

## 打包项目

`npm run build`
